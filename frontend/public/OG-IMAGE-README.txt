OPEN GRAPH IMAGE REQUIRED
========================

For optimal social media sharing, please create an image named "og-image.png" with these specifications:

Dimensions: 1200 x 630 pixels
Format: PNG or JPG
File size: Under 8 MB
Location: /public/og-image.png

The image should include:
- TempMail logo/branding
- Tagline: "Free Temporary Email Service"
- Clean, professional design
- High contrast for readability

This image will be displayed when your website is shared on:
- Facebook
- Twitter
- LinkedIn
- WhatsApp
- Slack
- Other social platforms

You can create this image using:
- Canva (canva.com)
- Figma (figma.com)
- Adobe Photoshop
- Any design tool

Once created, save it as: og-image.png in the /public folder.
